# Simple Groceries SQL Script

This repository contains a single, simple SQL script designed to create a basic table and insert sample data. It's intended as a straightforward example for initial SQL practice.

## Files

* `Simple_SQL.sql`: The main script that creates the `groceries` table and inserts four rows of data.

---

## Getting Started

### Prerequisites

You will need:
* Access to a running SQL database (e.g., MySQL, MariaDB, or PostgreSQL).
* The corresponding command-line client installed on your system.

### Usage

1.  Download the `Simple_SQL.sql` file.
2.  Open your terminal and navigate to the directory where the file is saved.
3.  Run the script against your desired database:

```bash
mysql -u [username] -p [database_name] < Simple_SQL.sql
